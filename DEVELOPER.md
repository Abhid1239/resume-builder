## Running in local

<br>

- Open the terminal / command prompt in your system

  <br>

- Clone the repo

  ```
  $ git clone https://github.com/sadanandpai/resume-builder.git
  ```

  <br>

- Install the dependencies

  ```
  $ yarn install
  ```

  or

  ```
  $ npm install
  ```

  <br>

- Run the project

  ```
  $ yarn dev
  ```

  or

  ```
  $ npm run dev
  ```

  <br>

- Open the url http://localhost:3000/ on your browser
